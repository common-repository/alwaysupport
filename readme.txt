== Description ==

AlwaySupport, provides a self-service customer support product, that reduce support costs, improve customer service, and increase online conversions. 
It does so by providing instant and contextually relevant answers based on a self-learning and ever growing Q&A knowledge-base.

== Installation ==

* Activate the plugin through the 'Plugins' menu in WordPress
* Get widget code from [Alwaysupport website](https://www.alwaysupport.com/BarWidget.html "Alwaysupport website") 
* Place this code in Alwaysupport plugin settings

== Frequently Asked Questions ==

= Why should you use Alwaysupport? =

It's all about increasing sales, retaining your valuable users and reducing support costs. By providing automatic assistance, you'll have a huge drop in the amount of support emails and tickets, your customers will get the assistance they need, you will retain more users and sales will increase.
